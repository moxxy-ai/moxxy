# moxxy

Block-based, modular agentic loop framework. Every block — provider, loop strategy, tool, compactor, even the CLI — is swappable.

**Status:** v0 in active development. See [`AGENTS.md`](./AGENTS.md) for the architecture overview and contribution guide.

## Quick start

```bash
pnpm install
pnpm build
pnpm test
```
